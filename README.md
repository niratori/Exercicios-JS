* Biblioteca `readline-sync` para interação via terminal.

## ⚙️ Como executar
1.  Clone esse repositório.
2.  No terminal da pasta raiz, execute: `npm install`
3.  Para rodar qualquer exercício, use: `node nome-do-modulo/ex01.js`

## 📝 Descrição do Projeto

Este sistema foi desenvolvido como parte de um projeto colaborativo para o curso de **JavaScript e Liderança em Tecnologia**. O objetivo é criar uma ferramenta eficiente para auxiliar no processo de memorização e estudo através de **Flashcards**.

O sistema permite a criação, organização e revisão de cartões de estudo, aplicando conceitos de lógica de programação, CRUD, manipulação de dados e modularização em JavaScript.

---

## 🚀 Funcionalidades 

- ✅ **Criação de Decks:** Organização de flashcards por temas ou categorias.
- ✅ **Gerenciamento de Cartões:** Adicionar, listar, editar e remover perguntas e respostas.
- ✅ **Sistema de Revisão:** Interface para o usuário testar seus conhecimentos.
- ✅ **Persistência de Dados:** Armazenamento das informações para uso contínuo.
